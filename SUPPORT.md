# Support

Thank you for using AutoMappic! We want to ensure you have the best experience possible.

## Getting Help

If you have questions about how to use AutoMappic or encounter issues, please use the following channels:

### Documentation
Before opening an issue, please check our [comprehensive documentation](./README.md) and the [docs/](./docs/) directory. Most common patterns and limitations are documented there.

### GitHub Discussions
For general questions, architectural advice, or sharing how you use AutoMappic, please use [GitHub Discussions](https://github.com/Digvijay/AutoMappic/discussions).

### GitHub Issues
If you've found a bug or have a feature request:
- Search existing [Issues](https://github.com/Digvijay/AutoMappic/issues) to see if it's already being tracked.
- If not, use our Bug Report or Feature Request templates in the issue tracker.

## Security Vulnerabilities
If you discover a security vulnerability, please follow our [Security Policy](./SECURITY.md). **Do not report security vulnerabilities via public issues.**

## Contributing
Interested in helping out? Check our [Contributing Guide](./CONTRIBUTING.md).

---

We aim to respond to all inquiries within a reasonable timeframe, but please remember that AutoMappic is an open-source project maintained by volunteers.
